# Personal Homepage

一个基于公开 GitHub 主页信息制作的静态个人主页，当前版本采用更克制、精致的 editorial / portfolio 风格设计。

## Files

- `index.html`: 页面结构
- `styles.css`: 页面样式
- `script.js`: 滚动显隐与基础交互
- `assets/avatar.jpg`: 头像资源
- `assets/favicon-rounded-64.png`: 标签页图标
- `assets/apple-touch-icon.png`: 移动端图标

## Preview

直接在浏览器中打开 `index.html` 即可预览，或在当前目录启动任意静态文件服务。
